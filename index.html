Como usar:

  • Clique nos botões + para aumentar o valor em 1
  • Clique nos botões - para diminuir o valor em 1
  • Clique no botão + Dia em cada dia da semana para aumentar todos os horários daquele dia
  • Os valores são limitados entre 0 e 100
  • Use "Exportar como JSON" para baixar os dados como arquivo JSON
  • Use "Exportar como Imagem" para baixar a tabela como PNG
  • Use "Importar Tabela" para carregar dados de um arquivo JSON
  • Use "Salvar no Navegador" para guardar os valores atuais
